# Archive

Archived material lives here because it still has historical value but is no longer part of the active execution path.

## Main Archive Areas

- `sdd-v1/`: the previous SDD framework, old feature packages, flat spec packs, manuals, and related legacy guidance
- `reviews/`: historical review reports
- `spec-results/`: historical generated artifacts
- `sdd/`: earlier archive history retained before the v2 refactor

Use archive material only when active docs explicitly require historical comparison.
